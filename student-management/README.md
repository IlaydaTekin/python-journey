# Student Management System

This is a simple Python-based student management system that allows users to store and analyze student information.

## Features
- Create student objects
- Store student names and grades
- Calculate average grades
- Display student details
- Basic object-oriented programming (OOP) structure

## Project Structure

- `main.py` → Main program logic  
- `student.py` → Student class definition  

## How it works

The program creates student objects using a class and processes their data.  
Each student has a name and a list of grades. The program calculates the average grade and displays the results.

## How to run

```bash
python main.py

Grades: [70, 80, 90]

Average: 80
