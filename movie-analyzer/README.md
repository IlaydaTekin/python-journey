# Movie Analyzer

This is a simple Python project that analyzes movie-related input data.

## Features
- Accepts user input
- Processes text data
- Extracts and analyzes information
- Displays results in a structured format

## How it works
The program takes movie-related input from the user and processes it step by step to generate meaningful output.

## How to run

```bash
python movie_analyzer.py

Enter movie name: Inception
Enter rating: 9

Output:
Movie: Inception
Rating: 9
Analysis: Highly rated movie
