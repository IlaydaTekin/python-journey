# CLI Calculator

This is a simple command-line calculator built with Python.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Continuous calculation loop
- Basic error handling (division by zero)

## How it works
The program displays a menu where the user selects an operation, enters two numbers, and gets the result.  
It continues running until the user chooses to exit.

## How to run

```bash
python calculator.py
